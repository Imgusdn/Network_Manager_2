## 실기
### 라우터
`확인`
```
인터페이스 정보
en - show interface - copy r s
접속한 사용자
en - show user(s) - copy r s
라우팅 테이블
en - show ip route - copy r s
플래쉬
en - show flash - copy r s
```
`변경`
```
호스트 네임을 ICQA로 변경
en - conf t - hostname ICQA - exit - copy r s
```
